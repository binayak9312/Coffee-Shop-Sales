# Coffee-Shop-Sales
Tools Used::: • Excel • Power Pivot • Data Model • DAX Measures

Coffee Shop Sales Dashboard (Excel | Power Pivot | Data Model)
📌 Project Overview

This project presents an interactive Coffee Shop Sales Dashboard built using Microsoft Excel.
The dashboard provides insights into sales performance, customer behavior, and product trends using Power Pivot, Data Model, and DAX measures.

It demonstrates how raw transactional data can be transformed into meaningful business insights.

🎯 Objectives
Analyze overall sales performance
Understand customer footfall trends
Identify top-performing products
Evaluate store-wise performance
Enable dynamic filtering using slicers
